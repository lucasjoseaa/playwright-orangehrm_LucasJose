# Automação com Playwright - OrangeHRM - Lucas José - QA Engineer

## 📌 Projeto
Automação de testes E2E em sistema de RH utilizando Playwright.

## 🚀 Tecnologias
- Playwright
- TypeScript

## ✅ Cenários testados

- Cadastro de funcionário
- Busca de funcionário
- Fluxo completo

npx playwright testgit add
